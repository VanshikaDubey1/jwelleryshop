# **App Name**: Shreeji Photobooks Online

## Core Features:

- Service Showcase: Display Shreeji Photobooks' services: Photo Printing, Album Printing, Acrylic Printing.
- Image Gallery: Showcase Shreeji Photobooks' work with albums, acrylic prints and wall frames. Users can upload their photos here to see how they look in the shop's gallery
- Booking Form: Allow customers to place orders, specifying service, size, quantity, uploading photos, choosing delivery/pickup, preferred date and additional notes, and saving data to Firestore.
- Order Management: Enable admin to manage bookings with status updates (Pending, Printing, Ready, Delivered) and search filter.
- Contact Form: Enable users to send messages.
- Email Notifications: Send email notifications to Shreeji Photobooks on new bookings using EmailJS.
- Order ID Generation: Automatically generate a unique Order ID upon booking submission and confirm to the customer in a thank you message.

## Style Guidelines:

- Primary color: Gold (#CBA13A) to reflect the premium quality and tradition.
- Background color: Light gray (#F5F5F6), a desaturated version of the primary color, to provide a clean, modern backdrop that keeps focus on content.
- Accent color: Charcoal (#111114), a dark neutral color that stands apart from gold, creates effective contrast on highlights, and lends a contemporary edge.
- Headings font: 'Poppins' (sans-serif) for a modern, clean title.
- Body font: 'Lato' (sans-serif) for readable, accessible text.
- Use elegant icons representing photo printing services, albums and contact details.
- Implement smooth fade and slide animations when transitioning between pages.
